Nikola gosta de categorizar tudo o que vê. Uma vez, Stephan deu-lhe uma máquina de etiquetar no seu aniversário, e os robôs foram espalhando etiquetas em toda a superfície do navio durante semanas. Ele já categorizou todos os reagentes em seu laboratório, livros na biblioteca e notas sobre a mesa. Mas, então, ele aprendeu sobre dicionários Python e categorizou todas as configurações possíveis para os drones da Sophia. Agora, os arquivos são organizados em uma profunda estrutura aninhada, mas não Sophia gosta disto. Vamos ajudar Sophia a achatar esses dicionários.

Dicionários python são tipos de dados para armazenar e processar configurações. Eles te permitem armazenar dados por chaves para criar estruturas aninhadas. Você terá um dicionário onde as chaves são ou string ou um dicionário. O objetivo é achatar o dicionário, mas preservar a estrutura nas chaves. O resultado deve ser de um dicionário sem os dicionários aninhados. As chaves devem conter o caminho que contém as chaves mães do dicionário original. As chaves no caminho são separadas por "/". Se o valor for um dicionário vazio, deve ser substituído por uma string vazia (""). Vejamos um exemplo:

Condições prévias:

  • Chaves em um dicionário não são strings vazias;
  • Valores em um dicionário são strings ou dicts;
  • root_dictionary != {}.
